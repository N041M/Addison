Bundled fonts — Fern design direction (Addison shell)
=====================================================

These woff2 files are bundled locally (never hotlinked) because the app runs
under a strict Content-Security-Policy (default-src 'self'). All three families
are licensed under the SIL Open Font License, Version 1.1 (OFL-1.1), which
permits bundling and redistribution with the software.

Each was subset to the `latin` + `latin-ext` character sets (so Czech and other
Central-European diacritics render in the serif body). Source: Google Fonts,
retrieved 2026-07-18 via the google-webfonts-helper API
(https://gwfh.mranftl.com), which serves the upstream Google Fonts binaries.

Each upstream project ships the complete OFL.txt with the reserved font names
and copyright notices above. These families are the one sanctioned exception to
the project's "system stacks only" rule: bundling keeps them CSP-safe (no
external host is ever contacted), and the @font-face stacks in styles.css always
fall back to system serif / sans / mono if a file fails to load.
